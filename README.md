# Google Drive to html

This is a quick POC that converts google drive into html

## Depenedency
1. Install Python
2. Install dependencies
    ```
    pip install google-api-python-client
    ```
3. Run the script
    `python3 script.py`
4. open the wiki page `wiki.html` in browser
